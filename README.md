# 🎥 Drishyam – AI Powered Smart CCTV Surveillance System

## 🚀 Overview
Drishyam is an AI-based smart CCTV system that performs real-time surveillance and detects suspicious activities using computer vision and machine learning.

## ❗ Problem
Traditional CCTV systems only record footage and require continuous human monitoring, leading to delayed responses and inefficiency.

## 💡 Solution
Drishyam analyzes live video feeds and detects:
- Unauthorized access
- Suspicious movements
- Potential threats

It sends real-time alerts, enabling rapid crisis response.

## 🛠 Tech Stack
- Python
- OpenCV
- Flask
- Machine Learning (extendable)
- Google Vertex AI (planned integration)

## ⚙️ Features
- Real-time video monitoring
- AI-based threat detection
- Live streaming via web interface
- Scalable architecture
